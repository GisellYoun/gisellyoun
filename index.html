ABOUT MYSELF

I am a front-end user-interface web developer with a computer engineering B.S. from Calpoly who grew up studying art. I have worked for large corporations such as Cisco, HP, Apple and Sun Microsystems. My career started out as a programmer at Apple but the nature of my work has evolved from application to user interface. Eventually, I switched my career entirely to user interface and design. I have built user interfaces using Apple Cocoa. When I switched to web development, I have taken additional courses to learn SQL database, object oriented javascript, Adobe FLEX and Adobe Flash Builder. I have worked with Perl, C, C++ and Java like any other programmers would. When building HP and Cisco websites, I have used techniques to import the master design of the company to propagate through the child pages, where variable elements are used to dynamically import the central designs. Now, I am studying graphic design at Berkeley Extension.
